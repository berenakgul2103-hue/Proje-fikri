# Lovable Yayin Adimlari

Bu proje statik bir HTML + JS uygulamasidir. Lovable ile hizlica yayinlayabilirsin.

## 1) Dosyalari Hazirla
- `index.html`
- `app.js`

## 2) Lovable Projesi Olustur
- Lovable uzerinde yeni bir proje ac.
- "Import project" veya "Upload files" secenegi ile bu dosyalari yukle.

## 3) Build/Run Ayari
- Bu proje icin ozel build gerekmez.
- Entry point olarak `index.html` sec.

## 4) Gemini API Key Ayari
- Guvenlik icin API key'i frontend koduna sabit yazma.
- Uygulamayi acinca sayfadaki `Gemini API Key` alanina key girerek calistir.
- Uretim ortami icin ideal yontem: backend proxy kullanmak (key'i sunucu tarafinda saklamak).

## 5) Publish
- "Publish" veya "Deploy" butonuna bas.
- Uretilen URL ile uygulamayi mobil ve masaustu cihazlarda test et.
